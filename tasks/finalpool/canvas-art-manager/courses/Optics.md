# Optics

Credits: 3

Learning Outcomes:
- Apply ray and wave optics to lens and mirror systems.
- Build and characterize a simple laser setup.

| Week | Topics              |
| ---- | ------------------- |
| 1    | Geometric Optics    |
| 2    | Lenses & Mirrors    |
| 3    | Optical Instruments |
| 4    | Wave Optics         |
| 5    | Interference        |
| 6    | Diffraction         |
| 7    | Mid-Term            |
| 8    | Polarization        |
| 9    | Laser Physics       |
| 10   | Fiber Optics        |
| 11   | Non-linear Optics   |
| 12   | Lab Mini-Project    |
| 13   | Design Review       |
| 14   | Lab Report          |
| 15   | Final Exam          |

Assessment: 30 % Labs, 25 % Mid-term, 30 % Final, 15 % Report.
Textbook: Hecht, Optics, 5e.
Policies: Safety goggles mandatory in lab.
